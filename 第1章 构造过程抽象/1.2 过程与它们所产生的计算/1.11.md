# 练习 1.11

函数 `f` 由如下的规则定义：如果 `n < 3`，那么 `f(n) = n`；如果 `n >= 3`，那么 `f(n) = f(n-1) + 2f(n-2) + 3f(n-3)`。请写一个采用递归计算过程计算 `f` 的过程，再写一个采用迭代计算过程计算 `f` 的过程。

---

递归版本：

```Scheme
(define (f-recursion x)
  (if (< x 3)
      x
      (+ (f-recursion (- x 1))
         (* 2 (f-recursion (- x 2)))
         (* 3 (f-recursion (- x 3))))))
```

迭代版本：

```Scheme
(define (f-iteration x)
  (f-iter 0 1 2 x))

(define (f-iter a b c x)
  (if (< x 3)
      c
      (f-iter b
              c
              (+ (* a 3) (* b 2) c)
              (- x 1))))
```
