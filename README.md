## 代码环境

本项目使用 [DrRacket](https://download.racket-lang.org/) 7.8 开发，依赖 sicp 插件。

安装好 DrRacket 后，使用 `raco` 安装 sicp 包。

```bash
raco pkg install --deps search-auto sicp
```
