# 练习 1.18

利用练习 1.16 和 1.17 的结果设计一个过程，它能产生一个基于加法、加倍和折半运算的迭代计算过程，只用对数的步数就能求出两个整数的乘积。

---

```Scheme
(define (multiply a b)
  (assert (and (integer? a) (> a 0)
               (integer? b) (> b 0)))
  
  (if (> a b)
      (multiply-iter a b 0)
      (multiply-iter b a 0)))

(define (multiply-iter a b prev-a)
  (if (= b 1)
      (+ a prev-a)
      (multiply-iter (+ (double a) prev-a)
                        (halve b)
                        (if (even-integer? b) 0 a))))

(define (even-integer? n)
  (= n
     (double (halve n))))

(define (double n)
  (+ n n))

(define (halve n)
  (quotient n 2))

; 60
(multiply 6 10)
```
